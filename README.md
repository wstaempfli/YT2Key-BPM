# YT2Key-BPM

Tools for downloading YouTube tracks and renaming them with detected musical key and BPM.

## Project parts

- CLI downloader/tagger: `scripts/playlist_key_bpm_downloader.py`
- Reusable core module: `scripts/downloader_core.py`
- Companion API service: `service/app.py`
- FL Studio VST3 plugin UI source: `plugin/`

- CLI usage and setup: `README_playlist_tool.md`
- VST3 + companion setup: `README_vst_companion.md`
